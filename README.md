# Outset

**Discover Unique Outdoor Experiences**

Outset is a web application that helps college students and young adults find and plan outdoor activities near them. Built with vanilla JavaScript, HTML, and CSS.

## Features

- 🔍 **Location-based Search** - Find outdoor activities by city and state
- 🎯 **Smart Filters** - Filter by activity type, duration, difficulty, and cost
- 📅 **Schedule Planning** - Add activities to your personal schedule
- ☀️ **Weather Integration** - Real-time weather data to plan your activities
- 👥 **Community Features** - Connect with other outdoor enthusiasts and plan group activities
- 🗺️ **Map Integration** - Get directions to any activity via Google Maps
- 🔗 **Share Activities** - Share your favorite spots with friends

## Activity Types

- Outdoor: Hiking, Biking, Kayaking, Climbing, Skiing, Camping, and more
- Indoor: Climbing gyms, Swimming pools, Fitness centers, Yoga studios, and more
- 60+ different activity categories

## Technologies Used

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **APIs**:
  - OpenStreetMap Nominatim (Geocoding)
  - OpenStreetMap Overpass (Location data)
  - Open-Meteo (Weather forecasts)
  - Google Maps (Directions)
- **Storage**: LocalStorage for user profiles and schedules

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Enter a city and state to start exploring!

## How to Use

1. **Search for Activities**: Enter your city and state on the landing page
2. **Browse Results**: View outdoor activities organized by type
3. **Apply Filters**: Narrow down activities by your preferences
4. **Add to Schedule**: Click "Add" on any time slot to schedule an activity
5. **Join Community**: Create a profile and connect with other users
6. **Create Group Activities**: Organize group outdoor adventures
